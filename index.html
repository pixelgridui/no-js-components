PIXEL · GRID · UI

shadcn, rebuilt with the platform.

Five components. Native HTML and CSS only. 47 KB of JS swapped for 4 KB of CSS.

01

Accordion

<details> + interpolate-size

Native expand/collapse. Keyboard, focus, and ARIA grouping come free. Smooth height animation via interpolate-size: allow-keywords.

02

Dialog

popover attribute + ::backdrop

The browser handles focus trap, Escape-to-close, and backdrop. Just popover on the element + popovertarget on the button.

03

Dropdown

popover + anchor-positioning

Menu anchors itself to the trigger with CSS anchor(). One line of JS handles the outside-click edge case.

04

Tabs

radio inputs + :has()

Radio inputs hold the selected state. :has() swaps the visible panel. Keyboard arrow-key navigation is automatic.

05

Tooltip

sibling-combinator + @starting-style

Hover or focus the trigger, sibling tooltip fades in. @starting-style + transition-behavior: allow-discrete animate it from display: none.

Companion code to the Pixel Grid UI long-form video on rebuilding shadcn/ui without JavaScript.

Each demo is a standalone .html + .css pair. Open the files. Read every line. Nothing hidden.
